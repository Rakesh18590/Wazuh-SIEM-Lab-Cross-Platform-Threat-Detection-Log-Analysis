R# SIEM Lab: Wazuh FIM & Brute Force Detection

## Overview
This project demonstrates a SIEM lab using Wazuh on Ubuntu Server and a Windows Server agent.

## Components
- Wazuh Manager (Ubuntu)
- Wazuh Indexer
- Wazuh Dashboard
- Windows Agent

## Use Cases
### File Integrity Monitoring (FIM)
Detected file changes in monitored directories.

### Brute Force Attack
Simulated multiple failed login attempts and detected them.
